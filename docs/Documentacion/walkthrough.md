# Equipo de Especialistas IA Integrado

He configurado un entorno de desarrollo de nivel empresarial dentro de este proyecto. Ahora tienes a tu disposición un equipo completo de agentes especializados instalados localmente en `.agent/skills/`.

## Miembros del Equipo

### 1. Gestión y Estrategia (CTO)
- **cto-orchestrator**: Optimización multi-agente y coordinación de tareas complejas.
- **c4-architecture**: Generación de diagramas y documentación de arquitectura de software.

### 2. Desarrollo Senior
- **senior-developer**: Especialista Fullstack con análisis de calidad de código.
- **javascript-pro**, **python-pro**, **java-pro**, **typescript-expert**: Maestría en los lenguajes principales.

### 3. Base de Datos y Supabase
- **supabase-specialist**: Automatización total de Supabase (queries, almacenamiento, edge functions).
- **db-migration**: Gestión de migraciones seguras y sin fallos.

### 4. Calidad y Testing (Garantía de Funcionamiento)
- **testing-playwright**: Automatización de pruebas en navegador (E2E).
- **testing-patterns**: Patrones de testing y TDD para un código libre de bugs.
- **clean-code**: Aplicación de estándares pragmáticos de código limpio.

### 5. Seguridad y Auditoría
- **security-hardening**: Endurecimiento de la seguridad de la aplicación y dependencias.
- **project-auditor**: Auditoría exhaustiva de seguridad y cumplimiento.
- **production-audit**: Escaneo de código línea a línea para preparación de producción.

### 6. DevOps y Herramientas Pro
- **github-actions**: Automatización de CI/CD.
- **committer**: Gestión profesional de commits bajo el estándar de Conventional Commits (corregido y optimizado para este proyecto).
- **ai-engineer**: Desarrollo avanzado de aplicaciones basadas en grandes modelos de lenguaje (LLMs).
